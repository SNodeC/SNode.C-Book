SNode.C source/editing package

This archive is intended for manuscript inspection and refinement. It contains the Markdown manuscript, proposal material, source-version pin, metadata, figures, filters, LaTeX support, and example source trees needed to inspect or rebuild the book sources. The full manuscript PDF is intentionally omitted from this source/editing package and must be rebuilt from the Markdown sources before publisher/reviewer distribution.

Included:
- pdf/book-proposal-package.pdf
- pdf/README-PDF.md
- proposal/
- chapters/
- parts/
- figures/src/
- figures/pdf/
- filters/
- latex/
- examples/
- metadata.yaml
- book-files.txt
- CMakeLists.txt
- README.md
- STRUCTURE.md
- SOURCE-VERSION.md

Deliberately excluded:
- pdf/snodec-book.pdf when stale or not rebuilt from the current Markdown
- .git and other VCS metadata
- IDE state such as .qtcreator, .idea, .vscode
- local build directories and CMake cache files
- generated LaTeX auxiliary files
- existing package archives

Final publisher/reviewer package policy:
- preserve SOURCE-VERSION.md in the generated publisher/reviewer package
- rebuild pdf/snodec-book.pdf from the current Markdown
- include pdf/snodec-book.pdf and pdf/book-proposal-package.pdf
- remove pdf/README-PDF.md if it only says the full PDF is omitted
