# Designing with SNode.C

MiniGateway has now shown how the book's protocol, configuration, deployment, and role vocabulary can be combined into a complete system. This final technical part steps back from that construction and names the judgment behind it.

The question is no longer only how to build the system. The question is how to choose boundaries that remain understandable when the system grows. Chapter 37 condenses the decision model. Chapter 38 then generalizes the extension discipline: adding capabilities without mixing responsibilities that should remain separate.
