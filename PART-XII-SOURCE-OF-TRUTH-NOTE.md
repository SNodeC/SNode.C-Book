# MiniGateway source alignment note

Chapter 37 is aligned with `examples/MiniGateway-Base`, which is the base source tree supplied by the author.

Chapter 38 is aligned with `examples/MiniGateway-Extended`, which extends the base application with a Unix-domain measurement input role. The chapter uses the extension as a concrete example, but its prose also discusses testing, debugging, deployment, constrained systems, process splitting, and safe extension of SNode.C applications more generally.

The source archives used for this pass were:

- `MiniGateway-Base.tar.gz`
- `MiniGateway-Extended.tar.gz`

These source trees are treated as the source of truth for Part XII.
