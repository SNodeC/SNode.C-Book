SNode.C publisher/reviewer package

This archive is intended for publisher/reviewer evaluation. It contains the generated article-style proposal PDF, the generated proposal-with-sample-chapters PDF, the generated full manuscript PDF, the exact SNode.C source-version pin, and the clean source material needed to inspect or rebuild the proposal/manuscript package.

Deliberately excluded:
- .git and other VCS metadata
- IDE state such as .qtcreator, .idea, .vscode
- local build directories and CMake cache files
- generated LaTeX/index auxiliary files
- existing package archives

The proposal-package target writes one stable archive name:

  dist/packages/snodec-book-proposal-package.tar.gz

Historic timestamped proposal-package archives are removed from dist/packages before
that archive is created, so CI/artifact upload never collects stale packages.
